Num83r5

This project was made using unity and c#

This project is aiming to be a more intuitive calculator. I simply called it Numbers.

With Numbers, you can drag around, split and combine numbers.

In theory someone would learn mor from this than a normal calculator.

When you split numbers they divide into units, tens, hundreds etc.

The keypad can provide you with new numbers.

Numbers is based on how I mentally visualise math sometimes.

This may not count as a game, but I'm still rather happy with it.

I also made font for this thing.

I really only needed it for the numbers,

but unity would get upset if it wasn't a full font.

So the numbers look pretty good.
